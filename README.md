## My home page :house_with_garden:

[visit online](https://shapkarin.me) :eyes:

Current website use React, Redux, Github API and axios-mock-adapter to mock some requests.<br />
The backgound is generated by pure JS with Canvas API.
<br />
You can have a look at <a href='https://github.com/shapkarin/shapkarin.me' target='_blank'>the source code</a>.
It's a bit <a href='https://github.com/shapkarin/shapkarin.me/issues' target='_blank' data-tip="known issues">legacy</a>.
<br />
At least I don't think that redux-saga is necessary
and it don't has any CSS methodology.

### Install dependencies:
```
npm i
```

### Run:
```
npm start
```
